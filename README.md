# Cricket Talent Identification & Performance Analytics System

## Overview

This project analyzes IPL ball-by-ball cricket data to identify and rank players based on their batting and bowling performances. A custom Talent Score is calculated using Batting Impact and Bowling Impact metrics to support player scouting and performance evaluation.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Google Colab

## Dataset

* IPL Ball-by-Ball Dataset
* Historical IPL player performance records

## Features

* Batting Impact Score Calculation
* Bowling Impact Score Calculation
* Talent Score Ranking
* Top Player Identification
* Performance Visualization

## Results

The system generates ranked lists of players based on their overall contributions and identifies top-performing cricket talents using data analytics techniques.
